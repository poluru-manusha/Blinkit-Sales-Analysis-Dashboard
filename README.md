# Blinkit-Sales-Analysis-Dashboard
1. Project Title / Headline
Blinkit Sales Analysis Dashboard | Power BI Business Intelligence Project

2. Short Description / Purpose
Developed an interactive Power BI dashboard to analyze Blinkit's sales performance, customer preferences, and outlet trends. The dashboard transforms raw sales data into actionable business insights, enabling better decision-making through data visualization and KPI tracking.

3. Tech Stack
Power BI – Dashboard Development & Visualization
Power Query – Data Cleaning and Transformation
DAX (Data Analysis Expressions) – KPI Calculations and Measures
Microsoft Excel/CSV – Data Storage and Source

4. Data Source
Blinkit sales dataset containing information on:
Product Categories
Sales Revenue
Outlet Types
Outlet Locations
Item Ratings
Outlet Establishment Years
Product Attributes

5. Features / Highlights
📊 Business Problem
Analyze sales performance across products, outlets, and locations to identify growth opportunities and improve business decisions.

📈 Dashboard Features
Total Sales, Average Sales, Number of Items, and Average Rating KPIs
Sales Analysis by Item Type and Product Category
Outlet-wise Performance Analysis
Sales Distribution by Outlet Size and Location Tier
Outlet Establishment Trend Analysis
Fat Content Analysis (Low Fat vs Regular)
Interactive Filters and Slicers for dynamic reporting

💡 Key Insights
Identified top-performing product categories and outlet types.
Compared sales across different outlet locations and sizes.
Analyzed customer preferences using rating and product data.
Revealed sales trends to support strategic business planning.

🎯 Skills Demonstrated
Data Cleaning & Transformation
Data Modeling
DAX Calculations
Interactive Dashboard Design
Business Intelligence & Data Visualization
Analytical Thinking and Insight Generation

Business Impact
Enabled quick monitoring of key sales KPIs through an interactive dashboard.
Helped identify high-performing products and outlet types contributing the most revenue.
Supported data-driven decision-making by highlighting sales trends and customer preferences.
Improved visibility into outlet performance across different locations and sizes.
Reduced manual reporting effort by consolidating sales metrics into a single dashboard.
